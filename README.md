# Linux-Handouts für IBB-Kurse

Dieses Repository enthält vom Dozenten bereitgestellte Linux-Handouts

## Verzeichnisstruktur

- *alias*: Alias-Definitionen

- *bin*: Shell-Skripte

- *slides*: Präsentationsfolien im PDF-Format

- *win-bat*: Windows Batch-Dateien
